# 2775번 - 부녀회장이 될테야
[문제 링크](https://www.acmicpc.net/problem/2775)

## ✅ 문제 요약
아파트 구조에서 k층 n호의 사람 수 구하기.  
규칙:  
> k층 n호 = (k-1층 n호) + (k층 (n-1)호)

## 💡 접근 방법
- DP 점화식: `dp[k][n] = dp[k-1][n] + dp[k][n-1]`  
- 초기값: `dp[0][i] = i`  
- 테이블 미리 계산 후 결과 바로 반환.

## 🧠 배운 점
- 2차원 DP 테이블 구현 패턴 익힘.  
- 점화식을 직접 세우고 반복문으로 채움.  
- 메모이제이션 없이도 반복문으로 충분.  

## ⚙️ 시간 복잡도
O(k × n)

## ⚠️ 주의할 점
- 0층은 n명 → 초기화 필수.  
- k, n 최대 14 → 배열 크기 고정 가능.

---

🪶 **정리:**  
- 핵심: DP 점화식  
- 포인트: 초기값 설정  
- 복습용 키워드: 2D DP, 점화식 세우기
